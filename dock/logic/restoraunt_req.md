**ВКУСНЫЕ МЕСТА ГОРОДА** 

**Логика интерфейса (согласно макету):**

* **Формат:** Список карточек заведений, расположенных в сетку (Grid).  
* **Состав карточки (сверху вниз):**  
  1. **Превью-изображение:** Блок под фото заведения в верхней части карточки.  
  2. **Название заведения:** Крупный текстовый заголовок под фото.  
  3. **Описание:** Информационный текст о заведении.  
  4. **Блок кнопок:** Две кнопки в ряд, для перехода к картам. 

**Требования к кнопкам и ссылкам:**

* **Типы кнопок:**  
  * **Яндекс Карты:** Оранжевый фон, текст «АДРЕС "ОТКРЫТЬ В ЯНДЕКС КАРТАХ"».  
  * **2 ГИС:** Зеленый фон, текст «АДРЕС "ОТКРЫТЬ В 2 ГИС"».  
* **Масштабируемость:** Если у заведения несколько точек, кнопок может быть несколько (в указаниях к карточкам будет указано сколько необходимо кнопок.).  
* **Реализация ссылок:** На этапе проектирования необходимо подготовить только места (слоты/переменные) для ссылок; сами ссылки будут вставлены в код вручную.

**Стилизация (согласно общим указаниям):**

* Карточки должны иметь скругленные углы и мягкие тени в стиле Apple.  
* Использовать палитру из общих указаний: основной фон `#F5F2F2`, акценты оранжевого (`#FEB05D`) и зеленого (подходящий по тону) для кнопок карт.
