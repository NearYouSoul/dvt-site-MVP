**ОБЩИЕ ТРЕБОВАНИЯ К ФРОНТЕНДУ**

1. **Логика:** Текстовые инструкции в файлах — приоритет для функционала. Основной файл отображения проекта это index.html, в нем должны быть подключены все файлы css и js в котором созданы все разделы сайта, такие как header, game_table, restoraunt_cards, menu, notes.
2. **Стиль:** Современный минимализм (Apple-style). Характеристики: много свободного пространства («воздух»), скругленные углы (12–16px), мягкие тени, плавные анимации появления элементов. 
3. **Цветовая палитра:**  
   * **Background (основной):** `#F5F2F2`  
   * **Blocks/Buttons (акценты):** Синий (`#5A7ACD`), Оранжевый (`#FEB05D`), Темный (`#2B2A2A`).  
   * **Дополнительно:** Разрешены градиенты в тон и использование прозрачности (Glassmorphism) для Header.  
4. **Адаптивность:** Mobile First. Сайт должен идеально выглядеть на смартфонах и корректно масштабироваться до Desktop.
5. **Стек:** Vanilla JS and Tailwind CSS via CDN only
